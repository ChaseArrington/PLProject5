Instructions for running the Parser

The file to be parsed is "12 days of xmas.txt"
The Parser itself is "XMAS_PLY.py"
To run:

1) Navigate to the folder "Parser" in the command line
2) In the command line type: cat 12\ days\ of\ xmas.txt | python XMAS_PLY.py
3) When run, the quantity of each gift recieved in the poem
"The Twelve Days of Christmas" should be displayed.
